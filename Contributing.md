Contributing to Omoikane
We welcome contributions to the Archive! To maintain the integrity of our Narrative Truth Engine, please follow these guidelines:

Lore Integrity: New lore added to data/Marathon or data/Cyberpunk must be sourced from canonical texts (terminals, sourcebooks, or official wikis).

Validation Logic: Any changes to the "Dumbass Trigger" in main.py must be accompanied by a test case in the demo/ folder to prevent logic regressions.

Pull Requests: Please use the provided Pull Request template to describe your changes and verify that the FastAPI server and Discord bot are still communicating correctly on port 8000.
